### ⚠️ 常见问题排查

#### 问题 1：页面无法加载
**可能原因**：
- Vite 开发服务器未启动
- 端口被占用

**解决方法**：
```bash
# 检查端口占用
netstat -ano | findstr :5173

# 重启开发服务器
npm run dev
```

#### 问题 2：API 请求失败
**可能原因**：
- 后端服务未启动
- 环境变量配置错误

**解决方法**：
```bash
# 检查后端服务状态
# 检查 .env 文件配置
```

#### 问题 3：类型错误
**可能原因**：
- TypeScript 配置过于严格

**解决方法**：
```bash
# 运行类型检查
npm run type-check

# 如果有错误，可以暂时放宽限制
# 编辑 tsconfig.json，设置 strict: false
```

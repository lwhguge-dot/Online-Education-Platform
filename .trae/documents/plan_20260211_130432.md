# 学生中心 UI/UX 专业优化实施方案

本方案旨在将学生中心 (`StudentCenter.vue`, `StudentDashboard.vue`, `StudentMyCourses.vue`) 的用户体验标准提升至与首页一致，重点解决无障碍、响应式和性能问题。

## 1. 视觉对比度与无障碍升级 (A11y)
**目标**：确保所有辅助文本清晰可见，且屏幕阅读器友好。
- [ ] **文本颜色替换**：在所有学生中心组件中，将 `text-shuimo/60`、`text-shuimo/50` 等低对比度样式替换为 `text-muted` (Slate 500) 或 `text-slate-600`。
- [ ] **图像与图标优化**：
    - 为 `StudentCenter.vue` 中的头像添加 `alt` 属性。
    - 为 `StudentMyCourses.vue` 中的课程封面添加动态 `alt` 属性。
    - 为所有装饰性 Lucide 图标添加 `aria-hidden="true"`。
    - 为侧边栏切换、播放等纯图标按钮添加 `aria-label`。

## 2. 响应式网格精细化 (Responsive)
**目标**：适配 iPad Mini 和折叠屏设备。
- [ ] **重构网格布局**：
    - **仪表盘 (`StudentDashboard.vue`)**：将统计卡片网格更新为 `grid-cols-1 tablet:grid-cols-2 fold:grid-cols-3 lg:grid-cols-4`。
    - **课程列表 (`StudentMyCourses.vue`)**：将课程网格更新为 `grid-cols-1 tablet:grid-cols-2 fold:grid-cols-3 xl:grid-cols-4`。

## 3. 交互与触摸体验 (Interaction)
**目标**：确保移动端操作无误触。
- [ ] **增大点击区域**：
    - 在 `StudentMyCourses.vue` 中，确保"继续学习"、"立即报名"等按钮在移动端具有 `min-h-[44px]`。
    - 检查 `StudentDashboard.vue` 空状态下的操作按钮尺寸。

## 4. 性能与渲染优化 (Performance)
**目标**：减少侧边栏动画带来的布局抖动。
- [ ] **侧边栏优化 (`StudentCenter.vue`)**：
    - 为 `<aside>` 和 `<main>` 元素添加 `will-change: width, margin` 属性，优化折叠动画性能。

## 5. 验证环节
- [ ] 验证 iPad 尺寸 (744px) 下的网格显示。
- [ ] 验证课程封面的 Alt 文本是否正确生成。
- [ ] 验证移动端按钮高度是否 >= 44px。
